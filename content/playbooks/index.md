# Playbooks

Step-by-step recipes for things I want to do the same way twice. Setting up a new Python research project, auditing a backtest for look-ahead, onboarding a new client site, scaffolding a Quartz wiki, defuddling a set of URLs into clean markdown — the boring, repeatable work that benefits from being written down once.

The target reader is future me. If anyone else finds them useful, good.
